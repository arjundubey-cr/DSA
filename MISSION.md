# Mission: DSA & LeetCode Interview Prep

## Why
Systematically build the pattern recognition and problem-solving fluency needed to pass
technical coding interviews at top-tier companies. Working through NeetCode's curated
problem set in Python.

## Success looks like
- Solve any NeetCode 150 problem without looking at solutions
- Recognize the right data structure / algorithm pattern within ~2 minutes of reading a problem
- Explain time and space complexity for every solution
- Implement clean, idiomatic Python under interview pressure

## Constraints
- Working in Python
- Following NeetCode 150 ordering (Arrays & Hashing → Two Pointers → Stack → ...)
- Lessons should be tightly scoped to one pattern or problem per session

## Out of scope
- Language theory, compiler internals
- Problems outside NeetCode 150 (for now)
- System design (until core DSA is solid)
