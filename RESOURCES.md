# DSA Interview Prep Resources

## Knowledge

- [NeetCode 150 Roadmap](https://neetcode.io/roadmap)
  The structured problem list being followed. Use for: determining what to teach next and in what order.

- [NeetCode YouTube](https://www.youtube.com/@NeetCode)
  High-quality video explanations for every problem on the roadmap. Reliable, interview-focused. Use for: primary source for each problem's optimal solution walkthrough.

- [Python Docs: heapq](https://docs.python.org/3/library/heapq.html)
  Official docs for Python's heap module. Use for: heap-based approaches to Top K problems.

- [Python Docs: collections.Counter](https://docs.python.org/3/library/collections.html#collections.Counter)
  Official docs for Counter. Use for: frequency counting shorthand.

- [Big-O Cheat Sheet](https://www.bigocheatsheet.com/)
  Quick complexity reference. Use for: verifying time/space complexity claims.

## Wisdom (Communities)

- [r/leetcode](https://reddit.com/r/leetcode)
  Community for problem discussion, interview debriefs, and approach comparison.

- [NeetCode Discord](https://discord.gg/ddjKRXPqtk)
  Active community following the same roadmap. Good for getting unstuck.
