Valid Sudoku

Three sets, one key trick. O(1) time and space.

The Problem

Given a 9×9 board, determine if it's a valid Sudoku. You only need to check the current state — the board doesn't need to be solvable.

A board is valid if:

  1. Each row has no repeated digits (1–9).
  2. Each column has no repeated digits (1–9).
  3. Each of the nine 3×3 boxes has no repeated digits (1–9).

Cells with '.' are empty — skip them.

Example

The board above is valid. Each row, column, and 3×3 box contains no duplicates.

The Pattern: Sets for Membership

Same tool you used for Group Anagrams and Contains Duplicate: sets for O(1) membership checks. But here you need three separate "seen" trackers — one per constraint type.

rows  = defaultdict(set)   # rows[r]   → digits seen in row r
cols  = defaultdict(set)   # cols[c]   → digits seen in col c
boxes = defaultdict(set)   # boxes[k]  → digits seen in box k

For each cell (r, c) with a non-empty value:

  1. Check if the digit is already in rows[r], cols[c], or boxes[key].
  2. If yes → return False.
  3. If no → add to all three sets.

The Key Insight: Box Index

Box key: (r // 3, c // 3) — integer division maps each cell to its 3×3 box.

The 9×9 grid has nine 3×3 boxes. You need a single hashable key that identifies which box a cell belongs to:

(0,0)
rows 0-2, cols 0-2
(0,1)
rows 0-2, cols 3-5
(0,2)
rows 0-2, cols 6-8
(1,0)
rows 3-5, cols 0-2
(1,1)
rows 3-5, cols 3-5
(1,2)
rows 3-5, cols 6-8
(2,0)
rows 6-8, cols 0-2
(2,1)
rows 6-8, cols 3-5
(2,2)
rows 6-8, cols 6-8

Cell (4, 7) → box key (4//3, 7//3) = (1, 2). That's the middle-right box. No extra math needed.

Full Solution

from collections import defaultdict

def isValidSudoku(board):
    rows  = defaultdict(set)
    cols  = defaultdict(set)
    boxes = defaultdict(set)

    for r in range(9):
        for c in range(9):
            val = board[r][c]
            if val == '.':
                continue

            box_key = (r // 3, c // 3)

            if (val in rows[r] or
                val in cols[c] or
                val in boxes[box_key]):
                return False

            rows[r].add(val)
            cols[c].add(val)
            boxes[box_key].add(val)

    return True

Complexity

Time: O(1) Space: O(1)

Board is always 9×9 = 81 cells. All operations are bounded by a constant.

Mental Trace

Walk through cell (0, 0) = '5':

1 Not '.' → proceed.
2 Box key = (0//3, 0//3) = (0, 0).
3 '5' not in rows[0], cols[0], boxes[(0,0)] → no conflict.
4 Add '5' to all three sets.

Now when we visit any other cell in row 0, column 0, or the top-left 3×3 box — if it's also '5', it's caught instantly.
