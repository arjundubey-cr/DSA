Counting Frequencies

Manual dict
count = {}
for n in nums:
    count[n] = count.get(n, 0) + 1
Explicit. Works anywhere. Use when you need dict comprehension control.
Counter shorthand
from collections import Counter
count = Counter(nums)
# count.most_common(k) → top k
# count[x] → freq of x (0 if absent)
O(n log k) for most_common. Fine in interviews unless O(n) required.

Top K Frequent — Three Approaches

Sort (naive)
sorted(count, key=lambda x: count[x],
       reverse=True)[:k]
Time: O(n log n)Space: O(n)
Min-heap
import heapq
heapq.nlargest(k, count, key=count.get)
# or maintain heap of size k manually
Time: O(n log k)Space: O(n+k)
Bucket sort ← O(n) optimal
freq = [[] for _ in range(len(nums) + 1)]
for n, c in count.items():
    freq[c].append(n)

res = []
for i in range(len(freq) - 1, 0, -1):
    for n in freq[i]:
        res.append(n)
        if len(res) == k:
            return res
Time: O(n)Space: O(n)
Key insight: frequency ≤ n, so use freq as index. Avoids sorting entirely.
