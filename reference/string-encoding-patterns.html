String Encoding Patterns

Arrays & Hashing · Reference Sheet

Length-Prefix Encoding (canonical)

Format: {len}#{content} — repeated per string
def encode(strs):
    return "".join(f"{len(s)}#{s}" for s in strs)

def decode(s):
    res, i = [], 0
    while i < len(s):
        j = s.index("#", i)       # find next # starting from i
        n = int(s[i:j])           # parse length
        res.append(s[j+1 : j+1+n])  # slice exactly n chars
        i = j + 1 + n             # advance past this chunk
    return res
Time: O(n) encode + O(n) decode · Space: O(n)

Why Naive Approaches Fail

Join on delimiter
",".join(strs)   # breaks if "," in strs
"#".join(strs)   # breaks if "#" in strs
No character is safe — strings can contain any char.
Fixed-width chunks
# pad all to length 50, slice on decode
# wastes space; fails on long strings
Impractical with variable-length strings.
